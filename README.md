## What is Rezible?

Rezible is a "mission control" platform for teams to assist in operating complex software systems. 
It provides opinionated tooling for humans and agents to understand, change & manage components of production with confidence. 
In doing so it builds a layered understanding of the system & makes it available when needed. 

### Current Status
Rezible is currently under heavy development, with no stable release available.
