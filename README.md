## What is Rezible?

Rezible is a platform for teams to coordinate & automate the operation of complex software systems. 
It provides opinionated workflows for humans and guardrails for their agents to allow every human operator to
understand, change & maintain increasingly complex systems. 

### Current Status
Rezible is currently under heavy development, with no stable release available.
